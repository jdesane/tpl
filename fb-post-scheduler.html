How to get your Page Access Token

  1. Go to developers.facebook.com/tools/explorer
  2. Select Tpl Collective App from the Meta App dropdown
  3. Change the token type dropdown from User Token to Tpl Collective (your page)
  4. Click Generate Access Token - approve any permissions it asks for
  5. Copy the full token that appears and paste it below
