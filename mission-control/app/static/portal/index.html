Prompt Templates
Listing Description
Write a compelling MLS listing description for a [bedrooms] bed, [bathrooms] bath home at [address] in [city]. Price: [price]. Key features: [features]. Tone: professional but warm. Include neighborhood highlights. Keep it under 250 words.
Lead Follow-Up Email
Write a follow-up email to a real estate lead named [name] who expressed interest in [buying/selling] in [area]. They previously mentioned [detail]. Keep it personal, not salesy. Include a soft call-to-action to schedule a call. Sign as [your name], [brokerage].
Social Media Post
Write a [platform] post for a real estate agent about [topic]. Tone: [professional/casual/educational]. Include 3-5 relevant hashtags. Keep it engaging and under [150/280] characters for the main hook. Add a call-to-action at the end.
Market Analysis
Create a market analysis summary for [city/neighborhood] real estate. Include: median home price trends, days on market, inventory levels, buyer vs seller market indicators, and a 3-month forecast. Format with bullet points. Note: use this as a template and fill in actual MLS data.
Recruiting Outreach DM
Write a short, casual DM to a real estate agent named [name] at [brokerage]. I want to introduce them to LPT Realty through TPL Collective. Mention that I'm an active agent (not a recruiter), highlight the math advantage, and invite them to a no-pressure 15-minute call. Keep it under 100 words. Don't be salesy.
